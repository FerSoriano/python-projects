# 🐍 Python Projects

Repository containing a collection of personal projects built with Python. Each project is organized in its own folder and can be explored or run independently.

## 🤖 Included projects

- [API_Spotify](API_Spotify/): A Spotify API client that demonstrates secure use of environment variables and OAuth-style HTTP requests to fetch music data.
- [Administrador_Gastos](Administrador_Gastos/): A personal finance helper for categorizing incomes and expenses, with utilities to manage and analyze budget items.
- [api_mongodb](api_mongodb/): A REST API built with FastAPI and MongoDB, designed to handle user data and demonstrate a MongoDB-backed Python web service.
- [create_extract_zip](create_extract_zip/): A lightweight command-line utility for creating and extracting ZIP archives with a simple `create` and `extract` workflow.
- [download_manager](download_manager/): A small download cleanup utility focused on removing or organizing files in a downloads folder.
- [gestor-notificaciones](gestor-notificaciones/): A Google Sheets-driven notification system for medication reminders, capable of checking active schedules and sending alerts.
- [movie_time](movie_time/): A web scraping and movie cataloging project that extracts film rankings from Wikipedia and stores them for analysis.
- [olympic_medals_2024](olympic_medals_2024/): A data ETL pipeline with database connectivity, designed to import and process Olympic medal records.
- [payment_calendar](payment_calendar/): A payment tracking and alert system integrating Google Calendar, Google Sheets, email notifications, and task management.
- [price_tracking](price_tracking/): A price tracking script that monitors item prices over time and stores historical data in Google Sheets for trend analysis.
- [video_downloader](video_downloader/): A web video downloader built with `pytube`, designed to download and organize videos from a URL.

## ℹ️ How to use this repository

1. Change to the project folder you want to inspect:

   ```bash
   cd <project_folder>
   ```

2. Read the project's `README.md` if available, or inspect the main script (`main.py`) to understand how it works.

3. For projects that require dependencies, create a Python virtual environment and install the required libraries.
